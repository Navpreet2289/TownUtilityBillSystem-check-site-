Web App "TownUtilityBillSystem" is developed by using C#/.NET MVC, Entity Framework, SQL, Razor, Bootstrap, AJAX, JSON, HTML, CSS, Google Maps APIs, highcharts.js.
This app helps to create, edit, store, analyze the data for the town utility service. 
Realized for such utilities: electricity, water, heating, gas.
The app manage the data from the database: 
-- 20 related tables,
-- 4 utility service (electricity, water, heating, gas),
-- 1002 buildings, 
-- 1062 customers,
-- 3262 meters,
-- 11 meter types (with different numbers, verification periods).
